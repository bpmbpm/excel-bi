## code
Браузерный JS  
Код должен быть хорошо читаемым человеку с низким уровнем знания js. Избегай стрелочных функций.  
Объем кода на файл не должен превышать 1500 строк на файл, при необходимости группируй в модули js по тематической направленности. Отдельно формируй модули с UI (в названии модуля будет {}_ui.js) и логику обработки (в названии модуля будет {}_logic.js)
