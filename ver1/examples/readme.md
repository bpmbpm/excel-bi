# Примеры датасетов для Excel BI

## Файлы примеров

| Файл | Описание | Строк | Колонок |
|------|----------|-------|---------|
| `example1_sales.xlsx` | Продажи магазина по месяцам, категориям и регионам | 30 | 7 |
| `example2_climate.xlsx` | Климатические данные по городам (температура, влажность, осадки) | ~156 | 7 |

---

## Пример 1: Продажи магазина (`example1_sales.xlsx`)

### Описание датасета

Данные о продажах розничного магазина за 6 месяцев (январь–июнь).

**Колонки:**
- `Месяц` — месяц продаж (текст)
- `Категория` — категория товара (Электроника, Одежда, Продукты)
- `Товар` — название товара (текст)
- `Количество` — количество проданных единиц (число)
- `Цена` — цена за единицу, руб. (число)
- `Выручка` — общая выручка, руб. (число)
- `Регион` — регион продаж (Москва, СПб, Казань)

---

### Пошаговая инструкция: загрузка датасета

1. Откройте файл `index.html` в браузере (двойной клик или через меню "Открыть")
2. Нажмите кнопку **"+ Загрузить данные"** в правом верхнем углу
   *или* перетащите файл `example1_sales.xlsx` в зону загрузки на странице "Данные"
3. В открывшемся окне нажмите на область загрузки и выберите файл `example1_sales.xlsx`
4. Дождитесь появления зелёного уведомления "Датасет успешно загружен (30 строк)"
5. В предпросмотре справа появится информация о колонках и первые 10 строк данных

---

### Пошаговая инструкция: создание графиков

#### График 1: Выручка по категориям (столбчатая диаграмма)

1. Нажмите **"Графики"** в верхней навигации
2. Нажмите кнопку **"+ Новый график"**
3. В секции **"Тип визуализации"** выберите **"Столбчатая диаграмма"** 📊
4. В поле **"Название графика"** введите: `Выручка по категориям`
5. В поле **"Ось X / Категории"** выберите колонку `Категория`
6. В секции **"Метрики (ось Y)"** поставьте галочку у `Выручка`
7. В поле **"Функция агрегации"** выберите `Сумма (SUM)`
8. Нажмите кнопку **"💾 Сохранить график"**

**Результат:** Столбчатая диаграмма с суммой выручки по трём категориям (Электроника, Одежда, Продукты).

#### График 2: Динамика продаж по месяцам (линейный график)

1. Нажмите **"+ Новый график"**
2. Выберите тип **"Линейный график"** 📈
3. Введите название: `Динамика выручки по месяцам`
4. Выберите ось X: `Месяц`
5. Отметьте метрику: `Выручка`
6. Агрегация: `Сумма (SUM)`
7. Нажмите **"💾 Сохранить график"**

#### График 3: Структура продаж (круговая диаграмма)

1. Нажмите **"+ Новый график"**
2. Выберите тип **"Круговая диаграмма"** 🥧
3. Название: `Доля выручки по регионам`
4. Ось X: `Регион`
5. Метрика: `Выручка`
6. Агрегация: `Сумма (SUM)`
7. Нажмите **"💾 Сохранить график"**

---

### Пошаговая инструкция: создание дашборда

1. Перейдите в раздел **"Графики"** (если ещё не там)
2. Для каждого созданного графика нажмите кнопку 📌 **"Добавить на дашборд"**
3. Нажмите **"Дашборды"** в верхней навигации
4. Все три графика появятся в сетке дашборда
5. Для изменения количества колонок используйте кнопки **1 / 2 / 3 / 4** в панели инструментов
6. Чтобы создать новый дашборд — нажмите **"+ Новый дашборд"** в боковой панели

## Пример 2: Климатические данные (`example2_climate.xlsx`)

### Описание датасета

Еженедельные климатические наблюдения за 2023 год по трём городам России.

**Колонки:**
- `Дата` — дата наблюдения в формате ДД.ММ.ГГГГ (текст/дата)
- `Город` — Москва, Санкт-Петербург или Новосибирск (текст)
- `Температура` — средняя температура, °C (число)
- `Влажность` — относительная влажность воздуха, % (число)
- `Осадки` — количество осадков, мм (число)
- `Давление` — атмосферное давление, мм рт. ст. (число)
- `Скорость ветра` — скорость ветра, м/с (число)

---

### Пошаговая инструкция: загрузка датасета

1. Откройте `index.html` в браузере
2. Нажмите **"+ Загрузить данные"** или перетащите файл `example2_climate.xlsx`
3. В диалоге выберите файл `example2_climate.xlsx`
4. После загрузки в предпросмотре увидите ~156 строк, 7 колонок

---

### Пошаговая инструкция: создание графиков

#### График 1: Среднегодовая температура по городам (горизонтальная диаграмма)

1. Перейдите в **"Графики"** → **"+ Новый график"**
2. Тип: **"Горизонтальная диаграмма"** ↔️
3. Название: `Средняя температура по городам`
4. Ось X: `Город`
5. Метрика: `Температура`
6. Агрегация: `Среднее (AVG)`
7. Нажмите **"💾 Сохранить"**

#### График 2: Изменение температуры в Москве (без агрегации)

1. **"+ Новый график"**
2. Тип: **"График с областью"** 📉
3. Название: `Температура в Москве 2023`
4. Ось X: `Дата`
5. Метрика: `Температура`
6. **Снимите галочку "Группировать данные"** — данные отображаются без агрегации
7. Нажмите **"💾 Сохранить"**

**Результат:** Плавный график температуры по всем записям с явным сезонным трендом.

#### График 3: Сравнение климатических показателей (радарная диаграмма)

1. **"+ Новый график"**
2. Тип: **"Радарная диаграмма"** 🕸️
3. Название: `Климатический профиль городов`
4. Ось X: `Город`
5. Метрики: выберите `Температура`, `Влажность`, `Скорость ветра`
6. Агрегация: `Среднее (AVG)`
7. Нажмите **"💾 Сохранить"**

#### График 4: Таблица данных

1. **"+ Новый график"**
2. Тип: **"Таблица"** 📋
3. Название: `Климатические данные (таблица)`
4. Ось X: `Дата`
5. Метрики: `Температура`, `Влажность`, `Осадки`
6. Нажмите **"💾 Сохранить"**

---

### Пошаговая инструкция: создание дашборда

1. Добавьте все четыре графика на дашборд кнопкой 📌
2. Перейдите в **"Дашборды"**
3. Установите **3 колонки** для удобного просмотра
4. Чтобы переименовать дашборд — нажмите **"+ Новый дашборд"** и введите своё название

---

## Советы по работе с приложением

- **Несколько датасетов**: можно загрузить несколько файлов — они отобразятся в боковой панели. Каждый график привязан к своему датасету.
- **Редактирование графика**: нажмите ✏️ на карточке графика для изменения настроек.
- **Удаление**: нажмите 🗑️ для удаления графика с подтверждением.
- **Работа офлайн**: приложение работает полностью в браузере. Интернет нужен только для первой загрузки библиотек (Chart.js, SheetJS). После можно работать офлайн.
- **Формат файлов**: поддерживаются `.xlsx`, `.xls` и `.csv`. При CSV файл будет прочитан как есть.
